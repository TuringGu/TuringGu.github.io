2019-03-28 15:08:53

光标移动及命令编辑

ctrl + u：删除光标所在行整条命令
ctrl + k：删除光标至结尾处的命令文本
ctrl + w：删除光标左边的一个单词（空格隔开的字符串）
ctrl + a：光标移动到命令开头
ctrl + e：光标移动到命令结尾
alt + f：光标向右移动一个单词
alt + b：光标向左移动一个单词

历史命令查询

histroy记录了执行的历史命令

![历史命令前的数字]：快速执行历史命令
ctrl + r：搜索执行过的命令
fc -l：显示部分历史命令

实时查看日志

tail -f [filename]：实时显示日志文件内容
less [文件名]、shift + f：实时显示日志文件内容
